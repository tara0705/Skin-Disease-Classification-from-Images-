# Skin-Disease-Classification-from-Images-
Skin Disease Classification using Deep Learning (CNN). This project uses the HAM10000 dataset to classify multiple skin conditions from dermoscopic images. It includes image preprocessing, model training, evaluation, and an optional web interface, enabling early detection and AI-assisted healthcare diagnostics.
## Dataset

This project uses the HAM10000 skin lesion dataset.

Dataset source:
https://www.kaggle.com/datasets/kmader/skin-cancer-mnist-ham10000

Due to size constraints, raw images are not hosted directly on GitHub.
If required, the dataset is provided as a ZIP via an external link.
